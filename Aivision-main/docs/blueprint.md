# **App Name**: FloraFind

## Core Features:

- Image Upload: Allows users to upload images of flowers from their device.
- Flower Classification: Uses the MobileNet model to classify the uploaded flower image offline and to determine its species.
- Prediction Display: Displays the top 3-5 flower predictions with their corresponding confidence scores.
- Accuracy Tips: Provides the user with tips on how to provide images that MobileNet can use more effectively to identify a flower

## Style Guidelines:

- Primary color: Soft lavender (#E6E6FA) to evoke a sense of tranquility and nature.
- Background color: Very light off-white (#F8F8FF), almost the same hue as lavender but heavily desaturated, to keep things airy.
- Accent color: Pale rose (#FFB8C6), an analogous color to lavender, for highlights and interactive elements to draw attention without overwhelming.
- Font: 'PT Sans', a modern-looking sans-serif for headlines and body text
- Use clear and simple icons to represent different flower types.
- Clean and minimalist layout with image preview and results displayed side by side.
- Subtle animations for loading states and result updates.
